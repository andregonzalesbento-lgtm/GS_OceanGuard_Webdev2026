# 🌊 OceanGuard — Web Development

Plataforma de monitoramento oceânico desenvolvida com HTML, CSS e JavaScript puro.

## 🚀 Como rodar

Baixe o projeto e abra o arquivo `index.html` no navegador. Não precisa de servidor.

## ✅ Funcionalidades

- Slideshow com 3 imagens sobre oceanos
- Quiz interativo com 10 perguntas
- Formulário de contato com validação
- 3 opções de tema de cor (Oceano, Natureza, Noite)
- Layout responsivo para mobile

## 🛠️ Tecnologias

- HTML5
- CSS
- JavaScript puro (sem frameworks)
